# SQL MCQ - Day 07 (Subqueries)

## 1. What is a subquery in SQL?

- A) A query inside another query  
- B) A query used to create tables  
- C) A query used for indexing  
- D) A query used for deleting rows  

<details>
<summary>Answer</summary>

**Correct Answer:** A) A query inside another query

</details>

---

## 2. Where can subqueries be used?

- A) SELECT clause  
- B) WHERE clause  
- C) FROM clause  
- D) All of the above  

<details>
<summary>Answer</summary>

**Correct Answer:** D) All of the above

</details>

---

## 3. A subquery that returns only one value is called?

- A) Single Row Subquery  
- B) Multi Row Subquery  
- C) Nested Join  
- D) Inline Query  

<details>
<summary>Answer</summary>

**Correct Answer:** A) Single Row Subquery

</details>

---

## 4. Which operator is commonly used with multiple-row subqueries?

- A) =  
- B) IN  
- C) BETWEEN  
- D) LIKE  

<details>
<summary>Answer</summary>

**Correct Answer:** B) IN

</details>

---

## 5. Which operator checks if a subquery returns any rows?

- A) EXISTS  
- B) ANY  
- C) ALL  
- D) IN  

<details>
<summary>Answer</summary>

**Correct Answer:** A) EXISTS

</details>

---

## 6. Which subquery refers to the outer query?

- A) Nested Query  
- B) Correlated Subquery  
- C) Inline Query  
- D) Derived Query  

<details>
<summary>Answer</summary>

**Correct Answer:** B) Correlated Subquery

</details>

---

## 7. Which operator compares a value with any value returned by the subquery?

- A) ANY  
- B) ALL  
- C) EXISTS  
- D) IN  

<details>
<summary>Answer</summary>

**Correct Answer:** A) ANY

</details>

---

## 8. Which operator compares a value with all values returned by the subquery?

- A) EXISTS  
- B) ANY  
- C) ALL  
- D) IN  

<details>
<summary>Answer</summary>

**Correct Answer:** C) ALL

</details>

---

## 9. What is another name for a subquery?

- A) Nested Query  
- B) Linked Query  
- C) Joined Query  
- D) Inline Table  

<details>
<summary>Answer</summary>

**Correct Answer:** A) Nested Query

</details>

---

## 10. Which clause is most commonly used with subqueries?

- A) WHERE  
- B) GROUP BY  
- C) ORDER BY  
- D) LIMIT  

<details>
<summary>Answer</summary>

**Correct Answer:** A) WHERE

</details>
