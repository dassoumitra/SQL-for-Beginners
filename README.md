## 📘 About

This repository provides a **step-by-step SQL learning roadmap** designed for beginners.

It includes:

- 📖 Concept notes  
- 💻 SQL query practice  
- 🧠 MCQ tests  
- 🎯 Interview preparation  
- 🚀 Mini SQL project  

The goal is to **build strong SQL fundamentals and practical database skills**.
