# SQL MCQ - Day 06

## 1. Which SQL command removes all rows quickly?

- A) DELETE  
- B) TRUNCATE  
- C) DROP  
- D) REMOVE  

<details>
<summary>Answer</summary>

**Correct Answer:** B) TRUNCATE

</details>


## 2. Which join returns only matching rows?

- A) INNER JOIN  
- B) LEFT JOIN  
- C) RIGHT JOIN  
- D) FULL JOIN  

<details>
<summary>Answer</summary>

**Correct Answer:** A) INNER JOIN

</details>


## 3. Which join returns all rows from the left table?

- A) RIGHT JOIN  
- B) LEFT JOIN  
- C) INNER JOIN  
- D) FULL JOIN  

<details>
<summary>Answer</summary>

**Correct Answer:** B) LEFT JOIN

</details>


## 4. Which join returns rows from both tables including unmatched rows?

- A) INNER JOIN  
- B) LEFT JOIN  
- C) FULL JOIN  
- D) SELF JOIN  

<details>
<summary>Answer</summary>

**Correct Answer:** C) FULL JOIN

</details>


## 5. Which operation combines results of two SELECT queries?

- A) JOIN  
- B) UNION  
- C) MERGE  
- D) LINK  

<details>
<summary>Answer</summary>

**Correct Answer:** B) UNION

</details>
