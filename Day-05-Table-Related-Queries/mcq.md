## 1. Which SQL statement is used to modify existing records in a table?

- A) MODIFY  
- B) UPDATE  
- C) CHANGE  
- D) ALTER  

<details>
<summary>Answer</summary>

**Correct Answer:** B) UPDATE

</details>


## 2. Which SQL statement is used to remove records from a table?

- A) DELETE  
- B) REMOVE  
- C) DROP  
- D) CLEAR  

<details>
<summary>Answer</summary>

**Correct Answer:** A) DELETE

</details>


## 3. What happens if the WHERE clause is omitted in an UPDATE statement?

- A) Only one row is updated  
- B) All rows are updated  
- C) No rows are updated  
- D) The query will fail  

<details>
<summary>Answer</summary>

**Correct Answer:** B) All rows are updated
